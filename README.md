# Saliency-model-inference-tool-for-Unity

---

## Requirements

- ```Python 3.11.1```

- ```CUDA 12.1 for GPU use```

## How to use the tool

- Copy the SaliencyTool folder the **Assets** folder of your Unity project. In the first salience prediction all missing dependencies will be installed.

## Models used

- [Image prediction model CNN-360]([DaniMS-ZGZ/Panoramic-CNN-360-Saliency: Code and models for "Panoramic convolutions for 360ยบ single-image saliency prediction" (github.com)](https://github.com/DaniMS-ZGZ/Panoramic-CNN-360-Saliency))

- [Video prediction model SST-Sal]([edurnebernal/SST-Sal: SST-Sal: A spherical spatio-temporal approach for saliency prediction in 360ยบ videos (github.com)](https://github.com/edurnebernal/SST-Sal))
